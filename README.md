# ZipVault Pro Version 1 - Install Instructions

1. Extract this ZIP file.
2. Keep "game title.zip" in the same folder as the app if ZipVault Pro needs it.
3. Run "ZipVault Pro.exe".
4. Test on copied files first.

Included files:
- ZipVault Pro.exe
- game title.zip
- zipvault_icon.ico
- zipvault_icon.png

Safety:
ZipVault Pro organizes files. Always test on copied files before using your main ROM collection.
